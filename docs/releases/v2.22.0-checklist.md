# karta 2.22.0 — release test-drive checklist

Karta 2.22.0 releases the **built-in Go stack packs**: `go-naming` and `go-htmx` join `minimalism` and the other built-ins under `skills/_shared/sme/`, pinnable per binder via `sme[]` and seedable into a consumer repo's `.karta/sme/`. The packs themselves landed on main 2026-07-15 (`5984580`, review fixes `4fbd308`) — eighteen minutes *after* the 2.21.0 bump — and sat unreleased for six days at the wrong version. That gap was caught during release verification of unrelated bench work, which is exactly the class of drift the bench's field-adoption lane exists to surface; this release closes it.

This is also the **first release gated by the framework bench**: the version-bump commit must ship a green full-gate results file stamped with the new version and the RC sha, enforced by `scripts/hooks/precommit_gate.py` (release block), or the commit is refused.

- [x] Green full-gate file at the RC sha staged with the bump (`benchmarks/results/gate/2026-07-21-v2.22.0-gate.json` — version-qualified name so the same-day 2.21.0 baseline artifact `2026-07-21-gate.json` is preserved, not clobbered).
